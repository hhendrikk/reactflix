#### Start local server for dist
```
yarn start
```
#### Start development
```
yarn dev
```
#### Test coverage
```
yarn test
```
#### Test watch coverage
```
yarn test:watch
```
#### Build dist
```
yarn build
```
#### Build dist with analizer
```
yarn build:analizer
```
#### Build dist replacing react with preact
```
yarn build:preact
```
#### Build dist replacing react with preact with analizer
```
yarn build:preact:analizer
```
